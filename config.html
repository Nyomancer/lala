lala config module¶

Config module

lala.config.get(key, default=None)[source]¶

Returns the value of a config option. The section is the name of the calling file.

If key does not exist and default is passed, the default value will be saved for later calls and returned.

Parameters:
  • key – The key to lookup
  • default – Default value to return in case key does not exist
lala.config.set(key, value)[source]¶

Sets the value of key. The section is the name of the calling file.
